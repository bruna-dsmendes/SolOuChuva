## 🌍 Visão Geral do Projeto

O **SolOuChuva** é uma aplicação web intuitiva e divertida, criada para simplificar a sua rotina ao fornecer previsões meteorológicas de forma rápida e clara. Desenvolvido como parte integrante do bootcamp da Generation, este projeto explora a fundo o desenvolvimento web, com um foco especial na aplicação prática de Inteligência Artificial (IA) para otimização de código e implementação de conceitos de *vibecoding*.

O objetivo principal é oferecer uma experiência de usuário agradável e eficiente, onde a consulta da previsão do tempo se torna uma tarefa descomplicada e até mesmo divertida, com mensagens de humor e um design amigável.

---

## ✨ Funcionalidades Principais

*   **Previsão em Tempo Real:** Obtenha dados meteorológicos claros e objetivos para a sua localização.
*   **Design Amigável:** Interface limpa, leve e visualmente agradável, pensada para uma excelente experiência do usuário.
*   **Performance Otimizada:** Construído com foco na eficiência do código para garantir um carregamento rápido e responsivo.
*   **IA Assisted (Vibecoding):** O código foi estruturado e otimizado com o auxílio de inteligência artificial, aplicando conceitos de *vibecoding* para um desenvolvimento mais intuitivo e eficaz.
*   **Geolocalização Automática:** Detecta automaticamente a localização do usuário para fornecer a previsão mais relevante.
*   **Seções Colapsáveis (Accordion):** Organização do conteúdo para facilitar a navegação e a visualização das informações.
*   **Atualização Automática Silenciosa (10 min):** Mantém os dados atualizados sem interromper a experiência do usuário.
*   **Compartilhamento via Web Share API:** Compartilhe facilmente a previsão do tempo com amigos e familiares.
*   **Cache Inteligente Offline (localStorage, TTL 30min):** Armazena dados localmente para acesso rápido e offline, melhorando a performance.
*   **Animação de Troca de Cidade (Fade):** Transições suaves e visuais ao mudar de localização.
*   **Modo Noturno Automático:** Adapta a interface para o modo noturno com base no pôr do sol real.
*   **Histórico de Buscas (localStorage):** Salva as últimas cidades pesquisadas para acesso rápido.
*   **UV, Umidade e Sensação com Dicas Inteligentes:** Informações adicionais sobre o clima com sugestões úteis.
*   **Look do Dia + Mensagens de Humor:** Um toque divertido e personalizado para a sua previsão.
*   **Previsão 7 Dias:** Visualize a previsão estendida para os próximos sete dias.

## 🛠️ Tecnologias Utilizadas

O projeto SolOuChuva foi desenvolvido utilizando as seguintes tecnologias:

*   **HTML5:** Para a estruturação semântica do conteúdo web.
*   **CSS3:** Para a estilização responsiva e moderna da interface do usuário.
*   **JavaScript (ES6+):** Para a lógica de programação, manipulação do DOM e consumo da API.
*   **API Open-Meteo:** Uma fonte de dados meteorológicos gratuita e de código aberto, utilizada para obter as informações de clima.

## 🚀 Como Rodar o Projeto Localmente

Para configurar e executar o projeto SolOuChuva em sua máquina local, siga os passos abaixo:

1.  **Clone o Repositório:**

    ```bash
    git clone https://github.com/bruna-dsmendes/SolOuChuva.git
    ```

2.  **Navegue até o Diretório do Projeto:**

    ```bash
    cd SolOuChuva
    ```

3.  **Abra o `index.html`:**

    Simplesmente abra o arquivo `index.html` em seu navegador web preferido. Não é necessário um servidor local para este projeto, pois ele é uma aplicação *client-side*.
