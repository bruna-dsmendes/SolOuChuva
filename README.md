## 🌍 O que é o SolOuChuva?
O **SolOuChuva** é uma aplicação web intuitiva criada para descomplicar a sua rotina! Com um design limpo e divertido, ele permite que você consulte a previsão do tempo de forma rápida. 

Este projeto foi desenvolvido como parte integrante da minha jornada no **bootcamp da Generation**, explorando a fundo o desenvolvimento web e a aplicação prática de IA como ferramenta de produtividade e **vibecode**.

---

## ✨ Funcionalidades
- 🌤️ **Previsão em Tempo Real:** Dados meteorológicos claros e objetivos.
- 🎨 **Design Amigável:** Interface pensada para ser leve e visualmente agradável.
- 🚀 **Performance:** Construído com foco na eficiência do código.
- 🤖 **IA Assisted:** Código estruturado com o auxílio de inteligência artificial para otimização e aprendizado.

---

## 🛠️ Tecnologias Utilizadas
- **HTML5** - Estrutura semântica.
- **CSS3** - Estilização responsiva.
- **JavaScript (ES6+)** - Lógica e consumo de API.
- **API Open-Meteo** - Fonte de dados meteorológicos.
